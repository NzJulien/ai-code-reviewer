# AI Code Reviewer

> Paste any code and get an instant AI-powered review with bugs, security issues, performance tips, and a quality score.

## Live Demo
https://nzjulien.github.io/ai-code-reviewer

## Features
- Quality score 0-100
- Bug detection with line numbers
- Security vulnerability scanning
- Performance issue flagging
- Style & best practice analysis
- Concrete fix suggestions for every issue
- What's good section (strengths)
- Copy report or export as Markdown
- Supports JS, Python, TS, React, HTML, CSS, Java, auto-detect

## Stack
- Vanilla HTML/CSS/JS
- Claude claude-sonnet-4-6 API
- Zero dependencies, zero backend
